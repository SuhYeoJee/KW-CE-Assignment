# 190318-출력문자 실습  
변수에 저장한 값을 8진수, 10진수, 16진수로 출력해보기.  

# 출력문자 (FormatCharacter)
|출력 문자|내용| |
|---|---|-|
|%d|10진수 정수|**d**ecimal|
|%o|8진수 정수|**o**ctal|
|%x|16진수 정수|He**x**adecinal|
|%u|부호없는 10진수 정수|**u**nsigned
|%f|float 소수|**f**loat|
|%lf|double 소수|**l**ong **f**loat|
|%e|소수|**e**표기법|
|%E|소수|**E**표기법|
|%c|문자|**c**har|
|%s|문자열|**s**tring|

%는 연달아 입력해서 출력 (%%)
